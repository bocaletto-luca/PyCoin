Key Features

Pure Python Core

Implement blockchain logic, Proof-of-Work, and transactions in under 300 lines of code.

Modular Consensus

Experiment with Proof-of-Work, integrate Proof-of-Stake, or design your own protocol.

FastAPI REST API

Query chain data, submit transactions, and trigger mining via a lightning-fast HTTP interface.

P2P Networking

Synchronize multiple nodes over WebSocket gossip for a true decentralized experience.

ECDSA Wallet

Generate and manage secp256k1 key-pairs for secure, signed transactions.

CLI Tooling

Manage wallets and run nodes effortlessly with a Typer-based command-line interface.

Block Explorer

Browse your blockchain in the browser—inspect blocks, transactions, and addresses.

Docker-Ready

Launch multi-node networks in seconds with Docker Compose for testing and demos.

How It Works

  1. Create Blockchain Core: Blocks store transactions, previous hash & nonce. Mining solves SHA-256 puzzles.
  2. Generate Wallet: ECDSA secp256k1 key-pairs secure your funds and sign transactions.
  3. Broadcast Transactions: Submit via REST or P2P—mempool holds pending transfers.
  4. Mine New Blocks: Use “/mine” endpoint or CLI to earn PyCoin reward and validate transactions.
  5. Sync Peers: Nodes gossip blocks & transactions over WebSocket for consensus.

Quickstart


# Clone the repo
git clone https://github.com/bocaletto-luca/pycoin.git
cd pycoin

# Setup environment
python3 -m venv .venv
source .venv/bin/activate     # macOS/Linux
.\.venv\Scripts\activate      # Windows

# Install dependencies
pip install -r requirements.txt

# Generate wallet
python cli.py create-wallet --password "YOUR_PASSWORD"

# Run P2P node
python pycoin/node.py

# Start API server
uvicorn pycoin.api:app --reload --port 8000
